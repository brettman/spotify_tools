# SQL Archive

This folder contains one-off SQL helper scripts used during development and debugging.

## Files

- **check_saved_cluster.sql** - Query to verify saved cluster data
- **delete_metal_cluster.sql** - Script to remove a specific test cluster
- **fix_description.sql** - Script to repair cluster descriptions
- **fix_saved_clusters.sql** - Script to fix cluster data issues
- **verify_table.sql** - Query to verify table structure

These scripts are kept for reference but are not part of the regular application workflow.
